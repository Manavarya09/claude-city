# Code of Conduct

## Our pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Standards

Welcoming and inclusive language. Respect differing viewpoints. Accept constructive criticism gracefully.

Unacceptable: harassment, insults, personal attacks, publishing private information.

## Enforcement

Report incidents to **manav@manavaryasingh.com**. Adapted from Contributor Covenant 2.1.
